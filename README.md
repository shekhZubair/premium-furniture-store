# 🪑 LuxeHome - Premium Furniture Store

A modern and responsive furniture store homepage built using **React.js** for the **MOH Technology Frontend Developer Internship Assessment**.

The project focuses on premium UI/UX design, reusable React components, responsive layouts, and clean frontend architecture.

---

## 🚀 Live Demo

🔗 Live Link: https://premium-furniture-store-flax.vercel.app/

---

## ✨ Features

- Responsive design for Desktop, Tablet, and Mobile
- Premium Hero/Banner section with CTA
- Product showcase with 10+ furniture items
- Reusable React components
- Product cards with:
  - Image
  - Name
  - Category
  - Price
- Search bar UI mockup
- Category filter UI
- Smooth scrolling and animations
- Modern responsive footer

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub

---

## 🤖 AI Assistance

AI tools were used as a supporting assistant during the development process, not for generating the complete project.

AI was used for:
- Brainstorming UI/UX ideas and design improvements
- Getting suggestions for component structure
- Debugging and resolving minor coding issues
- Improving responsiveness and styling approaches

The majority of the project was designed, coded, and implemented manually, including React components, layouts, styling, and overall functionality.
